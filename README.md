# FitFusion

FitFusion is a comprehensive fitness website offering workout programs, nutrition advice, and recovery tips.

## Features

- Custom workout planner
- BMI calculator
- Nutrition guides
- Recovery techniques

## Setup

1. Clone the repository
2. Open index.html in your browser

## Technologies Used

- HTML5
- CSS3
- JavaScript
